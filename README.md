# Addrevenue Google Tag Manager Template

This repository contains the Addrevenue Google Tag Manager custom template.

The template loads the Addrevenue tracking script and can be configured as either a base tracking tag or a conversion tag.

## Template file

The Google Tag Manager template is exported as:

```text
template.tpl
```

---

## GTM Template update process

```text
1. Export the updated GTM template from Google Tag Manager.
2. Rename the exported file to template.tpl.
3. Replace the existing template.tpl in the repo.
4. Update README.md with the new instructions.
5. Commit template.tpl and README.md.
6. Copy the commit SHA.
7. Add that SHA as the newest version in metadata.yaml.
8. Add changeNotes explaining the replacement.
9. Commit metadata.yaml.
10. Push to the default branch used by the Gallery.
```
